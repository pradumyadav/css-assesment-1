CSS can added to HTML document in 3 ways:

  • Inline - by using yhe style attribute inside HTML elementsl
  • Internal- by using a style element in the head section

  • The most common way to add CSS, is to keep the styles in external CSS files.However,in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing class vs ID, the diffrence is that CSS applies a style to multiple elements.ID, on the other hand,applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and it's used JavaScript.

In CSS,selectors are target a specific element or range of elements on a web page. Once an element has been targeted, a style or set os styles can be applied to the element.

There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
